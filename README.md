## ✨ 主な武器（機能）

### 1. 📊 スクリーニング機能（ログポース）
- **100点満点のスコアリング**でお宝銘柄を自動発掘
- **東証33業種を自動判定**（海域マップ）
- **日足・週足の切り替え**対応
- **高速モード/完全モード**切り替え（3分 vs 30分）
- **急落銘柄の自動除外**（嵐を回避）

### 2. 🔬 バックテスト機能（航海日誌）
- 過去6〜12ヶ月の航海記録を検証
- 月次リターン・勝率を自動計算
- グラフで視覚的に確認可能

### 3. 📈 銘柄比較機能（海図比較）
- 複数銘柄のチャートを重ねて表示
- 相対的な強さを一目で比較
- スクリーニング結果から直接選択可能

### 4. 📖 航海の手引き（詳細マニュアル）
- **アプリ内で完全マニュアルを閲覧可能**
- タブ形式でわかりやすく整理
- ワンピース風の楽しい説明

## ⚡ 高速航海 vs 完全装備

### 高速モード（軽装備・推奨）⚡

```
航海時間: 3-5分
取得データ:
 ✅ 業種（海域）- 全銘柄
 ✅ テクニカル指標（海流情報）
 ✅ スコア（お宝度）
 ❌ PER/PBR/配当（宝箱の中身）- 0表示
```

### 完全モード（完全装備）📊

```
航海時間: 15-30分
取得データ:
 ✅ 業種（海域）- 全銘柄
 ✅ テクニカル指標（海流情報）
 ✅ スコア（お宝度）
 ✅ PER/PBR/配当（宝箱の中身）- 取得可能銘柄
```

### 💡 効率的な航海術

```
1. 高速モード（5分） → 500銘柄からトップ50を発掘
2. 完全モード（3分） → 50銘柄だけ精密調査
合計: 8分で完了！まるでゴムゴムの実のスピード！
```

## ❓ よくある質問（FAQ）

### Q: PER/PBRが0になる（宝箱が空？）
**A:** 高速モードでは取得しません。完全モードで宝箱の中身も確認できます。

### Q: 完全モードが遅い（航海が長い）
**A:** 高速モードで絞ってから完全モードを使うと効率的です。ナミさんの航海術を真似よう！

### Q: 日足と週足、どちらがおすすめ？
**A:** 週足がおすすめ（穏やかな海、信頼性が高い）。初心者海賊は週足から！

**その他の質問は、アプリ内の「航海の手引き」をご覧ください！**
